# Application Personnage pokémon


Présentation:

Ce projet avait pour de objectif d'Afficher la liste via RecyclerView, de 
Récupérer des donnée via API reste et de stocker ses données dans notre application mobile .

Logiciel à installer:

ANDROID STUDIO.

Consignes Respectées

Ecran avec une liste d’éléments.
Ecran avec le détail d’un élément.
Appel WebService à une API Rest.
Stockage de données en cache

Architecture :
Singleton
Design Patterns
MVC
Principes SOLID
